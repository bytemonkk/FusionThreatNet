# Synthetic Cybersecurity Dataset

This directory contains the synthetic cybersecurity dataset used to evaluate the FusionThreatNet framework.

## Dataset Characteristics

- Total Samples: 10,000
- Mixed Numerical and Categorical Features
- Authentication Logs
- Security Events
- Network Activities
- System Monitoring Records

## Target Variable

| Value | Meaning |
|---------|---------|
| 0 | Normal Activity |
| 1 | Attack / Anomaly |

The dataset was generated for cybersecurity anomaly detection, risk assessment, and alert prioritization research.
